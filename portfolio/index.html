MetaMesh

I designed an ILP (Integer Linear Programming) model that produces a minimum-cost large-scale optical transport network that is fully restorable in the event of all possible dual span failures scenarios. I achieved up to 21% in reduction in capacity investment allowing savings in CAPEX and OPEX for service providers.

MetaMesh Max Availability

I designed an ILP (Integer Linear Programming) model that maximize the availability of a meta-mesh network by minimizing the number of non-restore working capacities (non-restore lightpahts) allocated in a optical transport network.
